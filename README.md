# JISwin2USMac

## 概要

JIS配列キーボードを使ったWindows環境をUS配列のMacライクに使えるようにするスクリプト

* 無変換キーの空打ちで IME を「英数」に切り替え
* 変換キーの空打ちで IME を「かな」に切り替え
* 無変換/変換キー + 他のキーで無変換/変換キーをCtrlキーとして扱う
* 半角/全角キーを`キーにするなどUS配列化

## 動作環境

* Windows10

## 使い方

JISwin2USMac.exe をダウンロード、実行 タスクトレイに常駐
